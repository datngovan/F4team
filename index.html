Voice assistant project

Our project ideas is to build a voice assistant with Python based on External Libraries. Which can respond and run code to do specific tasks. For example, when I say “Hello” the assistant can respond “Hi, can I have you?”.

The purpose of this project is to create a voice assistant that can do specific tasks which can be customized. We have a lot of voice assistant such as, Alexa and Seri. They can do most of the work like play music or turn on device if the users connect with the IoT. We want to specialize this assistant to do some work that an IT student have to do manually. The first function that I think about is the voice assistant can show 5 nearly events depend on Google calendar API. The second function we working on is that the assistant can clone the github file push, add, pull.
